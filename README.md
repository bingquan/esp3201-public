# ESP3201 — Public Colab Mini Assignments

Public mirror of the student-facing, runnable Colab material for ESP3201
(Machine Learning in Robotics & Engineering). Kept separate from the main
course repo so notebooks can be cloned/opened without authentication.

All runnable Colab material lives under `mini_assignments/`, one folder per
week.

Currently mirrored:

- `mini_assignments/week04_planner_tradeoff_study/` — Week 4, search under
  real constraints: `starter/notebooks/week04_planning_colab.ipynb` (CPU-only)
- `mini_assignments/week05_embodied_system_critique/` — Week 5, VLM
  hallucination probe: `starter/notebooks/week05_vlm_grounding_colab.ipynb`
- `mini_assignments/week06_video_world_model_study/` — Week 6, video
  world-model probe: `starter/notebooks/week06_video_world_model_colab.ipynb`
- `mini_assignments/week07_reward_hacking_diagnostics/` — Week 7, Q-learning
  and reward hacking: `starter/notebooks/week07_rl_colab.ipynb` (CPU-only)
- `mini_assignments/week09_agent_safety_case_colab/` — Week 9, probing an
  agent: `notebooks/week09_agent_colab-v3.ipynb` (current version; earlier
  `week09_agent_colab-v2.ipynb` and `week09_agent_safety_case_colab.ipynb`
  kept for reference), with the assignment brief (`README.md`), submission
  template, and grading rubric (`grading_rubric.md`)
- `mini_assignments/week10_trustworthy_ai_probe/` — Week 10, fairness
  recomputation, explainability local-vs-global, and a privacy
  membership-inference probe: `starter/notebooks/week10_trustworthy_ai_colab.ipynb`
  (CPU-only)

Open any notebook directly in Colab from its GitHub URL.

Answer keys are intentionally not included here — this repo carries what
students need to run the labs and submit, but not grading answers.
